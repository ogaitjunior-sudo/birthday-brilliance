# Birthday Brilliance

Interactive birthday text effects for `Happy Birthday` and `Feliz AniversĂ¡rio`.

## Tech Stack

- TanStack Router
- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Vercel static deployment

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deploy on Vercel

Use this folder as the Vercel project root:

```text
birthday-brilliance-main (1)/birthday-brilliance-main
```

Vercel is configured by `vercel.json` to run:

```bash
npm install
npm run build
```

The output directory is:

```text
dist
```

## Project Structure

```text
src/
  components/
    ui/                         Reusable UI primitives
  features/
    birthday-effects/
      components/               Birthday effects UI
      effects.ts                Effect types and phrase helpers
      effects-list.ts           Effect metadata and ordering
  hooks/                        Shared React hooks
  lib/                          Shared utilities
  routes/                       TanStack Router file routes
  router.tsx                    Router setup and error boundary
  styles.css                    Global theme, Tailwind setup, utilities
```

## Notes

- Use `npm` for this project because `package-lock.json` is present and current.
- `src/routeTree.gen.ts` is generated by TanStack Router and should not be edited by hand.
